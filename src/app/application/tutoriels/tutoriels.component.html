Mini tutoriel Angular 10

Tout d'abord, Angular est un Framework open source écrit en Javascript conçu par Google qui permet le développement d'applications Web, notamment les single page application. Le framework est basé sur une architecture de type MVC (Model View Controller), de sorte que les données, l’affichage et les opérations peuvent être séparés pour mieux gérer les responsabilités ce qui permet une maintenabilité accrue et améliore la coopération.

En effet, une application angulaire possède 2 objets parents; un objet de type Module et un de type Composant.

  • Module ⇒ Tour de controle qui permet de gérer les composants.
  • Composant ⇒ En gros, une page constitué de code HTML, Typescript et CSS.

Mini tutoriel Crud

L'acronyme informatique CRUD (Create Read Update Delete) est utilisé pour désigner les quatre opérations de base de manipulation de base de données. Autrement dit :

  • Céer
  • Lire
  • Mettre à jour
  • Supprimer

Chaque composant du CRUD peut être associé à des types de requête SQL et méthodes HTTP.

Commande SQL HTTP
Create INSERT POST
Read SELECT GET
Update UPDATE PUT
Delete DELETE DELETE

Mini tutoriel relations entre objets

Il existe 5 relation de base entre objets

  1. Association
  2. Agrégation
  3. Composition
  4. Heritage
  5. Interface (implements)

L'association signifie une relation sémantique.Il est persistant entre les deux classes.

L' agrégation est un cas spécifique d’association non symétrique exprimant une relation de contenance entre les deux classes.

La composition est une agrégation plus forte, ce qui signifie que les éléments ne peuvent appartenir qu'à une seule relation de composition

L'héritage, c'est des sous-classes qui vont hériter des propriétés d'une superclasse

L'interface définit la relation contractuelle garantie par la classe d'implémentaion dans laquelle la classe implémentée doit respecter certaines conditions.
